Docker Hub: docker pull ivoya/retask09

В зависимости от мощности машины запуск сервера и клиента может быть как почти мгновенным, так и занимать до 2-3 минут. Если localhost не отвечает, подождите.

Frontend: "тонкий клиент" Korolev поверх Akka HTTP Server + Akka HTTP Client (для запросов на сервер) (Scala)

Backend: Akka HTTP Server с единственным эндпоинтом - получением системного времени (Scala)

Фронтенд сервер развернут на порте 8080, бэкенд - на 8081. При запуске контейнера следует прокинуть порт 8080.

Код доступен в гитхабе RETask09/code/
